# EUR/CHF Ticker

Eine minimalistische Web-App, die den Euro-Referenzkurs der Europäischen Zentralbank (EZB) gegenüber dem Schweizer Franken anzeigt – mit interaktivem Chart der gesamten Historie seit 1999.

## Features

- 📈 Aktueller EUR/CHF-Kurs mit Veränderung zum Vortag
- 📊 Interaktives Liniendiagramm (Chart.js) seit 1999
- ⏱️ Zeitraum-Filter: 1M, 6M, 1J, 5J, Max
- 🌙 Dunkles, minimalistisches Design
- 📱 Responsive (Desktop & Mobile)
- 🔄 Automatische tägliche Aktualisierung via Cron Job

Hinweis: Die Daten zum Eurokurs gehören der EZB. 
Darüber, wie sie verwendet werden dürfen gibt die Seite [Haftungsausschluss und Urheberrecht](https://www.ecb.europa.eu/services/using-our-site/disclaimer/html/index.de.html) der EZB Auskunft.
